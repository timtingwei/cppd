# ex3_03
>对于string类的输入运算符处理空白字符:

在执行读取操作时，string对象会自动忽略开头的空白（即空格符/换行符/制表符等）并从第一个真正的字符开始读起，知道遇见下一处空白为止。

>对于getline函数处理空白字符:

getline函数从给定的输入流中读入内容，直到遇到换行符为止，所以空白符会被保留。
